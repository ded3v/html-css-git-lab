Resumo do mini projeto

Neste mini projeto, vamos construir um contador interativo usando HTML, CSS e JavaScript.

A ideia é criar uma página simples onde o usuário consiga:

aumentar o valor do contador;
diminuir o valor do contador;
zerar o contador;
dobrar o valor atual;
escolher o valor do passo;
ver se o número é positivo, negativo ou zero;
ver se o número é par ou ímpar;
ver uma mudança visual na tela conforme o valor do contador muda.
