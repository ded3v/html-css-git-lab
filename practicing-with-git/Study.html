HTML semântico

<header> → cabeçalho da página ou seção; geralmente contém logo, título e menu
<nav> → área de navegação com links importantes do site
<main> → conteúdo principal da página; parte central do site
<section> → divide o site em blocos organizados por assunto
<article> → conteúdo independente; muito usado para posts, cards e notícias
<footer> → rodapé da página; geralmente contém contatos e direitos autorais
<h1> até <h6> → títulos e subtítulos organizados por hierarquia/importância
<p> → parágrafo de texto; usado para conteúdos escritos
<a> → cria links clicáveis para páginas, seções ou sites externos
<img> → exibe imagens no site
<figure> → agrupa conteúdo visual como imagens, gráficos ou vídeos
<figcaption> → legenda descritiva do conteúdo dentro do figure
<div> → container genérico muito usado para layout e organização visual
<span> → estiliza pequenas partes de um texto sem quebrar linha
<ul> → cria lista não ordenada com marcadores
<ol> → cria lista ordenada numerada
<li> → representa cada item da lista
<button> → cria botões interativos clicáveis
<form> → estrutura de formulário para envio de dados
<input> → campo de entrada de dados; texto, senha, email etc.
<textarea> → campo maior para textos longos ou mensagens
<label> → descrição associada a inputs de formulário
<iframe> → incorpora conteúdos externos como YouTube, Google Maps etc.

CSS básico

color → muda a cor do texto
background-color → muda a cor de fundo
font-size → altera o tamanho da fonte
font-family → define a fonte do texto
margin → cria espaço externo entre elementos
padding → cria espaço interno dentro do elemento
border → adiciona borda ao elemento
border-radius → arredonda cantos
display: flex → cria layouts flexíveis e alinhamentos
justify-content → alinha elementos horizontalmente
align-items → alinha elementos verticalmente
width → define largura
height → define altura
hover → altera estilo quando mouse passa por cima
text-align → alinha texto

JavaScript inicial

console.log() → mostra informações no console do navegador
let → cria variável que pode mudar valor
const → cria variável constante
if → executa condição se algo for verdadeiro
else → executa alternativa da condição
function → cria funções reutilizáveis
addEventListener() → escuta eventos como clique e teclado
querySelector() → seleciona elementos HTML
innerHTML → altera conteúdo HTML via JavaScript
textContent → altera somente texto
click → evento de clique
array → lista de valores
for → cria repetições/loops
while → repete enquanto condição for verdadeira
return → retorna valores de funções

Comandos Git mais usados

Comando Função
git status Mostra o estado atual dos arquivos.
git diff Mostra exatamente o que mudou nos arquivos.
git add . Prepara as alterações para o commit.
git commit -m "mensagem" Cria um ponto salvo na história do projeto.
git push Envia os commits para o GitHub.
git pull Baixa alterações do repositório remoto.
git clone Copia um repositório remoto para sua máquina.
git branch Mostra ou cria branches do projeto.
git checkout Troca entre branches ou versões.
git merge Junta alterações de branches diferentes.
git log Mostra o histórico de commits.
git init Inicializa um repositório Git local.
git remote -v Mostra os repositórios remotos conectados.
git restore . Descarta alterações locais nos arquivos.

Checklist de revisão

  • Entender a diferença entre tag semântica e div.
  • Saber usar header, nav, main, section, article, aside e footer.
  • Entender margin, padding, border e box model.
  • Saber usar git status, git diff, git add, git commit e git push.
  • Entender o papel do README em um projeto.
